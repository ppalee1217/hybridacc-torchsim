HybridDataDeliverUnit (HDDU) 規格書 v2

1. 概述

HDDU 以 MMIO 控制 4 組 AGU、4 個 SPM 連接埠與 4 個 NoC 連接埠。本版本移除 DMA。

  • SPM[0/1/2] read → NoC[0/1/2] send
  • NoC[3] recv → SPM[3] write

2. Template 參數

template <int SPM_ADDR_BITS = 32, int NOC_TAG_BITS = 6, int DATA_BITS = 256>
class HybridDataDeliverUnit;
  • SPM_ADDR_BITS: SPM 位址寬度
  • NOC_TAG_BITS: NoC tag 寬度（NoC addr width = NOC_TAG_BITS + 1）
  • DATA_BITS: 資料寬度

3. Port 定義

3.1 時序/控制

名稱方向寬度說明
clkin1系統時脈
reset_nin1低有效重置

3.2 MMIO

3.3 SPM（4 ports）

訊號群：spm_addr[i] / spm_req[i] / spm_we[i] / spm_wdata[i] / spm_rdata[i] / spm_ready[i]

port用途行為

2PLIread
3PLOwrite

3.4 NoC（4 ports）

送出（0/1/2）：noc_out_data[i] / noc_out_addr[i] / noc_out_valid[i] / noc_out_ready[i]

接收（3）：noc_in3_data / noc_in3_addr / noc_in3_valid / noc_in3_ready

5. AGU Bank 對映

  • 0x000~0x0FF: AGU_PS
  • 0x100~0x1FF: AGU_PD
  • 0x200~0x2FF: AGU_PLI
  • 0x300~0x3FF: AGU_PLO

6. 核心行為

  • AGU0/1/2 產生地址，SPM read 後送到對應 NoC out port。
  • AGU3 與 NoC3 recv 握手後，資料寫入 SPM3。
  • NoC out 在 ready=0 時保持 valid/data/addr，不可遺失。
  • interrupt 在 done 或 err 時拉高。

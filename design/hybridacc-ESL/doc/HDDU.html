HybridDataDeliverUnit (HDDU) 規格書 v4

適用版本：simulator/include/Cluster/HybridDataDeliverUnit.hpp（四通道獨立平行管線 + VR NoC 介面）

1. 模組定位

HDDU 負責在 MMIO 控制下協調 AGU、SPM 與 NoC。核心目標是將 PS/PD/PLI/PLO 四個通道解耦，讓各通道可平行工作，不因單一通道背壓而阻塞其它通道。

  • PS/PD/PLI：SPM read -> NoC request（各自兩級管線）
  • PLO：NoC request（讀請求）-> NoC response -> SPM write（兩級管線）

2. Template 與常數

template <int SPM_ADDR_BITS = 32, int NOC_TAG_BITS = 6, int DATA_BITS = 192>
class HybridDataDeliverUnit;
  • NOC_ADDR_BITS = NOC_TAG_BITS + 1
  • DATA_BYTES = DATA_BITS / 8
  • FIFO_DEPTH = 16（所有內部 FIFO 固定深度，無軟體可設定欄位）
  • NoC addr 編碼：[NOC_TAG_BITS-1:0]=tag，[NOC_TAG_BITS]=ultra

4. MMIO Map

4.1 AGU bank window

  • 0x000~0x0FF: AGU_PS
  • 0x100~0x1FF: AGU_PD
  • 0x200~0x2FF: AGU_PLI
  • 0x300~0x3FF: AGU_PLO

4.3 位址換算（ComputeCluster 視角）

  • HDDU_BASE = 0x1000
  • bank_base = HDDU_BASE + bank * 0x100
  • reg_addr = bank_base + reg_offset

範例：AGU_PD.REG_BASE_ADDR = 0x1000 + 0x100 + 0x00 = 0x1100；AGU_PLO.REG_CTRL = 0x1000 + 0x300 + 0x20 = 0x1320。

4.4 建議最小設定序列

  1. REG_BASE_ADDR
  2. REG_ITER01 / REG_ITER23
  3. REG_STRIDE0（必要時加 STRIDE1/2/3）
  4. REG_TAG_BASE / REG_TAG_STRIDE0
  5. REG_MASK_CFG
  6. REG_CTRL（bit0=1 啟動）

5. 行為規格（重點）

5.1 PS/PD/PLI（各自獨立，純組合邏輯）

  • Stage 0：AGU 產生 addr/tag/ultra；read_noc_addr_wait_fifo 未滿時對 SPM 發讀取請求。
  • Stage 0.5：SPM Response 到達後，pop read_noc_addr_wait_fifo，將 {data, noc_addr} push noc_req_fifo。
  • Stage 1：從 noc_req_fifo 送往對應 NoC 介面；握手成功：TX_PKT+=1，TX_BYTE+=DATA_BYTES。

無 Cooldown 機制。FIFO 深度固定為 FIFO_DEPTH=16。

5.2 PLO（獨立）

  • Stage 0：AGU tag/ultra 從 NoC bypass FIFO 送往 noc_plo_out；write_addr_fifo 未滿時，握手完成 push addr 並回應 AGU ready。NoC 請求握手也遞增 TX_PKT。
  • Stage 1：noc_plo_in response 與 write_addr_fifo pop 配對， push spm_req_fifo；累加 RX_BYTE+=DATA_BYTES。
  • Stage 2：從 spm_req_fifo 對 SPM port 3 發出寫入請求。
  • Stage 2.5：seq_process 監聽 SPM 寫入回應，若非 SPM_OK 則設定 ERR_CODE=SPM_ERROR。

5.3 Stall 條件

  • PS/PD/PLI：read_noc_addr_wait_fifo 已滿 或 SPM port 未 ready
  • PLO：write_addr_fifo 已滿 或 NoC noc_plo_out.ready_in=0
  • 所有 AGU 中 gen_valid=1 && gen_ready=0 的次數在每 cycle 累加至 COUNTER_STALL

5.4 中斷語意

interrupt = STATUS.bit4 (ERROR) || STATUS.bit2 (DONE)

6. 驅動範例

// helper
void hddu_wr(uint32_t base, uint32_t off, uint32_t data);
uint32_t hddu_rd(uint32_t base, uint32_t off);
void agu_wr(uint32_t base, int bank, uint32_t off, uint32_t data);

const uint32_t H = 0x1000; // cluster HDDU base

// configure AGU banks (example)
agu_wr(H, 0, 0x00, ps_base);
agu_wr(H, 1, 0x00, pd_base);
agu_wr(H, 2, 0x00, pli_base);
agu_wr(H, 3, 0x00, plo_base);

hddu_wr(H, 0x808, 0xF);      // enable planes (default is already 0xF)
hddu_wr(H, 0x800, 1u << 1); // CTRL_START (bit1): start all AGUs

while (true) {
    uint32_t st = hddu_rd(H, 0x804);
    if (st & (1u << 4)) break; // bit4 = err
    if (st & (1u << 2)) break; // bit2 = done
}

uint32_t tx_pkt = hddu_rd(H, 0x828); // COUNTER_TX_PKT
uint32_t tx_b   = hddu_rd(H, 0x82C); // COUNTER_TX_BYTE
uint32_t rx_b   = hddu_rd(H, 0x830); // COUNTER_RX_BYTE
uint32_t stall  = hddu_rd(H, 0x834); // COUNTER_STALL

// stop all AGUs: CTRL_STOP = bit2 → hddu_wr(H, 0x800, 1u << 2);
// soft-reset (clear FIFO/errors): CTRL_RESET = bit0 → hddu_wr(H, 0x800, 1u << 0);

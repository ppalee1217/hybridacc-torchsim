# HybridAcc Python Behavioral Simulator

## Overview
This directory contains the source code for the HybridAcc Python Behavioral Simulator, a tool designed to simulate and analyze the behavior of hybrid accelerator architectures. The simulator provides a flexible framework for modeling various components and interactions within hybrid systems.

## Components
- **Core Simulation Engine**: The heart of the simulator that manages the execution of simulation events
- **Non-Linear Models**: Modules that define the behavior of non-linear components within the hybrid accelerator
- **Configuration Files**: YAML files that allow users to customize simulation parameters and settings
- **Utility Scripts**: Helper scripts for running simulations, processing results, and visualizing data

## Getting Started
To get started with the HybridAcc Python Behavioral Simulator, follow these steps:
1. **Install Dependencies**: Ensure you have Python 3.x installed along with the required libraries listed in `requirements.txt`.
2. **Configure Simulation**: Edit the configuration files in the `config/` directory to set up your simulation parameters.
3. **Run Simulation**: Use the provided scripts in the `scripts/` directory to execute simulations and generate results.
4. **Analyze Results**: Utilize the analysis tools included in the simulator to interpret and visualize the simulation output.
